实验器材:
	潘多拉STM32L4开发板
	
实验目的:
	进一步学习ES8388和STM32 SAI功能的使用，实现一个简单的WAV录音机.
	
硬件资源:
	1.	指示灯RGB
	2.	KEY0、KEY1和KEY_UP按键
	3.	串口1
	4.	TFTLCD（板载）
	5.	TF卡(放置图片文件)
	6.	SPI FLASH（需要存放字库）
	7.	ES8388音频解码芯片（板载）
	8.	STM32L4的SAI音频接口（STM32L4内嵌）
	9.	耳机接口（自备耳机）
	10.	MIC咪头（板载）

	
	
实验现象:
	本实验开机后，先初始化各外设，然后检测字库是否存在，如果检测无问题，再检测TF卡根目录是否存在RECORDER文件夹，
	如果不存在则创建，如果创建失败，则报错。在找到TF卡的RECORDER文件夹后，即进入录音模式（包括配置ES8388和SAI等），
	此时可以在耳机（或喇叭）听到采集到的音频。KEY0用于开始/暂停录音，KEY2用于保存并停止录音，WK_UP用于播放最近一次的录音。
	当我们按下KEY0的时候，可以在屏幕上看到录音文件的名字、码率以及录音时间等，然后通过KEY2可以保存该文件，同时停止录音（文件名和时间也都将清零），
	在完成一段录音后，我们可以通过按WK_UP按键，来试听刚刚的录音。RGB_B(蓝灯)用于提示程序正在运行，RGB_R(红色)用于提示是否处于暂停录音状态。

	
注意事项:
	1.	本例程需要大家自行准备TF卡进行测试，录音文件放在TF卡根目录下的RECORDER文件夹下。
	2.	本例程需要用到耳机来收听录音文件，请大家自备耳机
	3.	本例程中串口1的波特率为115200
	4.	如果SPI FLASH没有中文字库请先按照第三十三章例程更新字库
